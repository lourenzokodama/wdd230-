Design Principles

Clean Design

Google Fonts API

fonts.google

Google Fonts API effectively uses white-space, creating a clean and calm design, making content clear and easy to understand.

This design is achieved by separating fonts into boxes with appropriate spacing, simplifying the presentation.

Rule of Thirds

Brigham Young University - Idaho

byui.edu

Following the "Rule of Thirds," the focal point (woman's face) is on the right, and text (another focal point) is on the left.

This alignment enhances focus on the image and readability of the text, improving user experience.

Alignment

GitHub

github.com

Six logos at the bottom are aligned both vertically and horizontally, simplifying understanding. Horizontal centering guides reading direction.

Alignment is a fundamental design principle that enhances page aesthetics and user appeal when used effectively.
